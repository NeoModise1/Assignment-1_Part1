# Cybersecurity Awareness Chatbot

## Project Overview
This project is a C# console-based Cybersecurity Awareness Chatbot. The application interacts with users and provides useful information about staying safe online. It covers topics such as phishing, malware, password security, safe browsing, VPNs, and two-factor authentication.

The chatbot is designed to simulate a real conversation and promote awareness of cybersecurity risks.

---

## Objectives
The objective of this project is to:
- Educate users about cybersecurity threats  
- Provide quick and simple security tips  
- Demonstrate C# programming and object-oriented concepts  
- Create an interactive console application

## How It Works
The program starts by launching the chatbot and asking the user to enter their name. After validation, the chatbot begins a conversation with the user. It processes input using keyword matching and provides relevant cybersecurity responses. If the chatbot does not understand the input, it suggests topics to ask about. The program runs continuously until the user types "exit".
